# tiny-games

set of 4 nano-sized games and few tools. tried to not make files bigger than 1300 bytes (if github used quic it'd be single packet per game/tool, actually).

and so:
- games:
  - ./q.html (first game with such size which was made, sort-of an abstract, simplified shooter similar to space invaders - requires keyboard)
  - ./r.html (racing game, similar to chrome dino)
  - ./t.html (procedurally/random generated space sim which adapts itself to device limitations)
  - ./u.html (mix of tilt maze and snake - requires gyro support... like the one you have in your phone)
 
- tools (all support dark mode natively):
  - ./a.html (basic spreadsheet app)
  - ./b.html (app for drawing math formulas on canvas)
  - ./c.html (small IDE/text editor with html preview and ai-like autocompletion)
  - ./d.html (basic todo/list app)
  - ./e.html (basic paint app)
  - ./f.html (simplified version or IDE. whole app/site fits in 210 bytes)
